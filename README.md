# 🍱 校园食堂菜单展示网页

一个简洁美观的校园食堂菜单展示网页，支持分类筛选功能。

## ✨ 功能特性

- **分类筛选**：支持全部、早餐、午餐、晚餐、小吃五个分类
- **响应式布局**：使用 CSS Grid 实现自适应卡片布局
- **白底简约风格**：清爽的白色背景配合橙色主题色
- **交互效果**：卡片悬停上浮动画、按钮状态切换

## 📁 项目结构

```
food-menu/
├── index.html    # 页面结构
├── style.css     # 样式文件
├── script.js     # 交互逻辑
└── README.md     # 项目说明
```

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可查看效果，或使用本地服务器：

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 使用 Node.js 启动本地服务器（需安装 http-server）
npx http-server -p 8000
```

然后访问 `http://localhost:8000` 查看效果。

## 🎨 设计风格

- **配色方案**：白底 + 橙色主题 (#e67e22)
- **字体**：Microsoft YaHei
- **卡片阴影**：柔和的阴影效果
- **圆角设计**：12px 圆角
